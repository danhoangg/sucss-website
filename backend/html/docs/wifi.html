WiFi

To perform attacks against WiFi networks, you need the aircrack-ng toolkit. This is comes included in Kali but can also be installed on most Linux systems.

If you're not using Kali, you'll likely want to install the toolkit, like so:

# apt-get update
# apt-get install aircrack-ng

You will likely need a dedicated USB WiFi adapter unless the one in your computer support monitor mode. Check the aircrack wiki for supported chipsets.

If using Kali in a VM you will need a USB WiFi adapter which needs to be passed through to your VM as a USB device.
