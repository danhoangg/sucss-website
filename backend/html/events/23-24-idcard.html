RFID Hacking

Date : 2023-12-06

Difficulty : Intermediate

Delivered By : Skyler Mansfield

Overview

This week we'll be taking a look at RFID and NFC technologies that power most mainstream access control systems and ID cards. We'll show you how to clone some of the most common cards, why it works even with cards that should be secure, and even let you use some kit to try it out yourselves!

Prerequisites

No prior knowledge is required for this session, however a basic understanding of the software we'll be using will go a long way!

  • Proxmark III Iceman software
  • Pre-built binaries
