'To Trusted Hardware (and back again?)' (Arm Guest Talk)

Date : 2022-10-26

Difficulty : n/a

Delivered By : Hugo Vincent from Arm

Overview

Hugo Vincent (Lead Security Research Architect - Arm) is joining us to talk about confidential compute and trusted hardware! Join us to learn about how we can secure data in use, establish trust in remote hardware, and use cryptography to forgo the need for trusted hardware entirely.
