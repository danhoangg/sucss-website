Networking 101

Date : 2023-11-15

Difficulty : Beginner

Delivered By : Matthew Grove

Overview

This week we will be covering the basics of network and WiFi security - showing you how to (legally!) infiltrate and attack the networks that surround you. All are welcome and the session won't rely on any prior knowledge. Several of the challenges are brand-new for this year (exciting!) so everyone should be able to learn something new.

Please note that most of this week's challenges will only be available in-person during the session.

Prerequisites

  • A Kali (or similar) VM downloaded and set up (we can help with this during the session, however it takes a long time to download).
  • A basic knowledge of Bash.
  • (Idealy) At least one USB-A port on whatever machine you use (for WiFi adapters).
